# twitter-clone


A **full-stack social media application** inspired by Twitter, built using the **MERN stack** with **Tailwind CSS** and **TanStack Query** for efficient data fetching and caching.  
Users can post tweets, edit or delete them, and explore others' posts â€” all with a smooth, responsive, and futuristic UI.

---

## Features

-  User Authentication (Signup/Login/Logout)
-  Create, Read, Update, Delete Tweets
-  View and interact with posts in real time
-  Fully responsive design for desktop and mobile
-  Optimized data management using **TanStack Query**
-  Dark mode with glassmorphism and futuristic design elements
-  Organized folder structure for scalable development

---

## Tech Stack

**Frontend:**
- React.js  
- Tailwind CSS  
- TanStack Query (React Query)

**Backend:**
- Node.js  
- Express.js  
- MongoDB with Mongoose  
- JWT Authentication  
- bcrypt.js for password hashing
